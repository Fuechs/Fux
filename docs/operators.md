# Operators

## Structure

## Assignment

Value | Description
-|-
`=` | [link](./objects.md#variables)
`===` | [link](./types.md#constants)
`++ <obj>` | increment and return value
`<obj> ++` | return value and increment
`-- <obj>` | decrement and return value
`<obj> --` | return value and increment
`<obj> += <val> ` | increment variable by a value
`<obj> -= <val> ` | decrement variable by a value
`<obj> *= <val> ` | multiply variable by a value
`<obj> /= <val> ` | divide variable by a value
`<obj> ^= <val> ` | sets variable to the power of value
`<obj> %= <val>` | sets variable to modulus of value
`<obj> \|= <val>` | sets variable to bitwise and of value
`<obj> &= <val>` | sets variable to bitwise or of value
