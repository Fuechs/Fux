An beautiful alternative for C++

Syntax

Fux has a flexible Syntax similar to Python but still preserves the possibilities of C++, which makes writing code in Fux approximately 40% faster than in C++.

Typing

If required, Fux provides several different object types per standard, with even more defined in the core package. And if speed is not your main focus you can ignore types completely.

Performance

The average Fux program executes faster than Python / C and has almost equivalent syntax.
